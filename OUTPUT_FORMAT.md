# 📊 Sports Monitor - 输出格式规范 v3.0.7

## 概述

本文档定义 Sports Monitor 的标准输出格式，确保所有赛事类型都以统一、清晰、信息完整的方式呈现。

**版本**: v3.0.7  
**生效日期**: 2026-03-11  
**适用范围**: 所有体育赛事输出

---

## 🎯 核心原则

1. **表格化** - 所有赛事必须使用表格形式输出
2. **完整性** - 每场比赛必须包含：时间、对阵、赛事名称、直播平台
3. **一致性** - 所有运动种类使用相同的格式结构
4. **可读性** - 适合终端、Discord、微信等多种平台

---

## 📐 表格格式详细规范

### 终端表格（ASCII）

使用 ASCII 字符绘制表格，兼容性最佳：

```
┌──────────┬─────────────────────────────┬──────────────┬────────────┐
│  列 1    │          列 2               │    列 3      │   列 4     │
├──────────┼─────────────────────────────┼──────────────┼────────────┤
│ 数据 1   │         数据 2              │   数据 3     │  数据 4    │
└──────────┴─────────────────────────────┴──────────────┴────────────┘
```

**字符说明**:
- `┌ ┐ └ ┘` - 四角
- `┬ ┴ ├ ┤ ┼` - 连接符
- `─` - 横线
- `│` - 竖线

### Discord/微信表格（简化版）

由于平台限制，使用简化格式：

```
【时间】08:00
【对阵】Celtics vs Mavericks
【赛事】NBA 常规赛
【直播】CCTV5、腾讯体育
─────────────────────────
【时间】11:30
【对阵】Lakers @ Warriors
【赛事】NBA 常规赛
【直播】腾讯体育、咪咕
```

## 🏀 各运动种类输出规范

### 篮球 (Basketball)

**必需字段**:
- 时间（北京时间 HH:MM）
- 对阵（客队 @ 主队 或 客队 vs 主队）
- 赛事名称（NBA 常规赛/CBA 季后赛等）
- 直播平台
- 场馆（可选）

**示例**:
```
┌──────────┬─────────────────────────────┬──────────────┬────────────┬──────────────┐
│  时间    │         对阵信息            │   赛事名称   │  直播平台  │    场馆      │
├──────────┼─────────────────────────────┼──────────────┼────────────┼──────────────┤
│ 08:00    │ Celtics @ Mavericks         │ NBA 常规赛   │ CCTV5      │ TD Garden    │
│ 19:35    │ 广东宏远 vs 辽宁本钢        │ CBA 季后赛   │ 咪咕体育   │ 东莞银行中心 │
└──────────┴─────────────────────────────┴──────────────┴────────────┴──────────────┘
```

### 足球 (Football)

**必需字段**:
- 时间（当地时间 + 北京时间）
- 对阵（主队 vs 客队）
- 赛事名称（英超/西甲/中超等）
- 直播平台
- 比赛状态（未开始/进行中/已结束）

**示例**:
```
┌──────────┬─────────────────────────────┬──────────────┬────────────┬──────────────┐
│  时间    │         对阵信息            │   赛事名称   │  直播平台  │    状态      │
├──────────┼─────────────────────────────┼──────────────┼────────────┼──────────────┤
│ 20:00    │ Man City vs Liverpool       │ 英超第 29 轮  │ 咪咕体育   │ 未开始      │
│ 22:00    │ Real Madrid vs Barcelona    │ 西甲第 28 轮  │ CCTV5      │ 未开始      │
│ 19:35    │ 上海海港 vs 山东泰山        │ 中超第 3 轮   │ 腾讯体育   │ 进行中 65'  │
└──────────┴─────────────────────────────┴──────────────┴────────────┴──────────────┘
```

### 羽毛球 (Badminton)

**必需字段**:
- 时间（北京时间）
- 对阵/项目（男单/女双等）
- 赛事名称（全英公开赛等）
- 直播平台
- 轮次（预赛/1/8 决赛/半决赛/决赛）

**示例**:
```
┌──────────┬─────────────────────────────┬──────────────┬────────────┬──────────────┐
│  时间    │         对阵信息            │   赛事名称   │  直播平台  │    轮次      │
├──────────┼─────────────────────────────┼──────────────┼────────────┼──────────────┤
│ 17:00    │ 陈雨菲 vs 安洗莹 (女单)     │ 全英公开赛   │ 优酷体育   │ 1/4 决赛    │
│ 19:30    │ 刘圣书/谭宁 vs 贾一凡 (女双)│ 全英公开赛   │ 优酷体育   │ 半决赛      │
└──────────┴─────────────────────────────┴──────────────┴────────────┴──────────────┘
```

### 网球 (Tennis)

**必需字段**:
- 时间（北京时间）
- 对阵（选手 A vs 选手 B）
- 赛事名称（印第安维尔斯等）
- 直播平台
- 轮次

### 电竞 (Esports)

**必需字段**:
- 时间（北京时间）
- 对阵（战队 A vs 战队 B）
- 赛事名称（LPL 春季赛等）
- 直播平台
- BO 系列（BO3/BO5）

**示例**:
```

│  时间    │         对阵信息            │   赛事名称   │  直播平台  │   赛制       │

└──────────┴─────────────────────────────┴──────────────┴────────────┴──────────────┘
```

- 时间（北京时间）
- 大奖赛名称
- 赛事阶段（练习赛/排位赛/正赛）
- 直播平台
- 赛道名称

**示例**:
```

│  时间    │         赛事信息            │   阶段       │  直播平台  │    赛道      │

│ 18:00    │ 澳大利亚大奖赛              │ 排位赛       │ CCTV5      │ 阿尔伯特公园 │
```

## 🔥 推荐指数显示规范

**星级评分** (基于 0-100 分):

| 分数范围 | 星级 | 说明 | 显示 |
|---------|------|------|------|
| 80-100 | ⭐⭐⭐⭐⭐ | 必看经典战 | 高亮 + 置顶 |
| 60-79 | ⭐⭐⭐⭐ | 值得关注 | 正常显示 |
| 40-59 | ⭐⭐⭐ | 普通比赛 | 正常显示 |
| 20-39 | ⭐⭐ | 一般比赛 | 降级显示 |
| 0-19 | ⭐ | 不推荐 | 隐藏或折叠 |

**显示位置**: 联赛标题行右侧

**示例**:
```
🏀 NBA - 今日 7 场比赛 | 热门程度：⭐⭐⭐⭐⭐ (85/100)
```

## 📺 直播平台映射表

### 国内主流平台

| 平台 | 适用赛事 | 备注 |
|------|---------|------|
| CCTV5 | NBA、足球、F1、奥运会 | 央视体育 |
| CCTV5+ | 次要赛事 | 央视体育赛事 |
| 腾讯体育 | NBA、中超、电竞 | 需会员 |
| 咪咕体育 | 足球、CBA | 需会员 |
| 优酷体育 | 羽毛球、网球 | 部分免费 |
| B 站 | 电竞、部分足球 | 免费 |
| 抖音 | 部分赛事集锦 | 免费 |
| 央视频 | 央视内容回放 | 免费 |

### 平台显示规则

1. **多个平台**: 使用顿号分隔 `CCTV5、腾讯体育、咪咕`
2. **付费标记**: 需会员的平台标注 `腾讯体育 [会员]`
3. **免费标记**: 完全免费的可标注 `[免费]`
4. **未知**: 未获取到平台信息显示 `待定`

## ⚠️ 特殊情况处理

### 1. 无比赛日

```
──────────────────────────────────────────────────────────
🏀 NBA - 今日无比赛
──────────────────────────────────────────────────────────
📌 今日无 NBA 赛程安排
💡 提示：明天有 5 场比赛，使用 --tomorrow 查看
```

### 2. 数据获取失败

```
──────────────────────────────────────────────────────────
🏸 羽毛球 - 数据获取中...
──────────────────────────────────────────────────────────
⚠️  暂时无法获取实时数据，请稍后重试
📌 最近已知赛事：2026-03-10 全英公开赛 1/8 决赛
```

### 3. 时间敏感提示

```
【即将开始】15 分钟后：Celtics @ Mavericks (08:00)
【进行中】第 3 节还剩 5:32：Lakers 78 - 82 Warriors
【已结束】Lakers 105 - 98 Celtics
```
